# Token Killer — Telegram Bot Security Assessment Tool

**Версия:** 1.0  
**Автор:** Senior Python Developer

---

*Кратко:* `Token Killer` — консольная утилита для демонстрации уязвимостей Telegram-ботов и тестирования их устойчивости (валидация токена, сбор информации о боте, работа с webhook, нагрузочное тестирование, демонстративные операции изменения профиля и отправки сообщений). Инструмент предназначен **только для легального тестирования** (тестирование собственных ботов или при наличии письменного разрешения владельца).

## Описание
`Token Killer` помогает безопасно и удобно:
- проверить валидность Telegram Bot token;
- собрать сведения о боте (getMe, webhook info, профиль);
- управлять webhook (просмотр / установка / удаление);
- проводить нагрузочные тесты API (с измерением латентности);
- отправлять сообщения (spam demo) с rate-limit-aware поведением;
- модифицировать профиль бота (setMyName / setMyDescription);


---

## Возможности (кратко)
- Валидация токена (`--validate`)
- Сбор информации о боте (`--info`)
- Webhook: просмотр / установка / удаление (`--webhook-info`, `--set-webhook`, `--delete-webhook`)
- Нагрузочное тестирование (`--load-test` с `--requests` и `--concurrency`)
- Демонстративная отправка сообщений (`--spam --chat-id <id> --count <n>`)
- Демонстративная модификация профиля (`--new-name`, `--new-desc`)

python -m venv .venv
